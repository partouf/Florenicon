
Florenicon
originally by Kwast @ Draenor-EU


Overview
--------
Lists anyone in your vicinity who gains and loses a Druid's Efflorescence buff.

Shows amount of healing from the Efflorensence "unit" (no that's not a typo), and colours green if it heals.
If all healing's gone to overheal the name stays white.


Addons required:
- None
